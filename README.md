# Build a GraphQL Server with Node.js and MongoDB

Build a GraphQL Server with Node.js and MongoDB tutorial. Learn how to setup a GraphQL Server from scratch using Node.js and Express. I also show you to setup babel-node for es7, nodemon for the server to auto reload, and mongoose for connecting to a MongoDB.
